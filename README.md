# personal-site

Personal homepage for Gyumin Lee (KRIBB postdoc, synthetic biology + agentic engineering).

## Stack
- Astro 4 + TypeScript + Bun
- Tailwind CSS + shadcn-style component conventions
- React islands for interactive (Cmd+K, AI Q&A, mini-demos)
- Cloudflare Pages hosting + Pages Functions for AI Q&A backend
- GitHub Actions: daily Google Scholar sync + auto-deploy

## Develop
```bash
bun install
bun run dev      # http://localhost:4321
bun run build
bun run preview
```
