About

Postdoctoral researcher at KRIBB Synthetic Biology Research Center. PhD work focused on systems-level dissection of methanol stress adaptation in Methylorubrum extorquens, combining adaptive laboratory evolution with multi-omics integration to identify engineering targets for industrial strain development.

Current postdoc work expands toward methanotroph metabolic engineering and biofoundry workflow automation. Build software in parallel: KURO (AI-driven primer design), PrimerBench (primer evaluation), and kuma (research utility) reduce manual cloning and analysis cycles for the lab.

The making of these tools follows a deliberate AI-augmented coding practice — Claude Code as the primary editor, structured prompts as the source of truth, generated TypeScript and Python as the artifact. Repositories on GitHub demonstrate the pattern, this site is one of them.
